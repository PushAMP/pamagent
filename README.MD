pamagent
========

This is a Python Agent for PAM platform.

How-to install
==============

Linux 64-bit
------------
    pip install pamagent
Other OS
--------
1. Install *rust-nightly* via [rustup](https://www.rustup.rs)
2. pip install --no-binary :all: pamagent
